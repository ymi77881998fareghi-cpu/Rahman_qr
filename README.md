# مؤسسه توانبخشی و نگهداری دختران کم‌توان ذهنی رحمان

یک صفحه وب مدرن، واکنش‌گرا و قابل نصب (PWA) برای نمایش اطلاعات مؤسسه و دریافت کمک‌های مردمی از طریق QR Code.

---

## امکانات

- طراحی Glassmorphism
- طراحی واکنش‌گرا
- انیمیشن‌های مدرن
- پس‌زمینه متحرک
- کپی شماره کارت
- کپی شماره حساب
- کپی شماره شبا
- دکمه تماس
- دکمه واتساپ
- دکمه مسیریابی
- دکمه پرداخت آنلاین
- قابلیت نصب مانند اپلیکیشن (PWA)
- Service Worker
- پشتیبانی از زبان فارسی (RTL)

## تنظیم اطلاعات بانکی

در فایل **index.html** مقادیر زیر را تغییر دهید:

### شماره کارت

```html
58946388000098845
```

### شماره حساب

```html
265430616
```

### شماره شبا

```html
IR610130100000000265430616
```

## تنظیم شماره تماس

در فایل index.html

```html
href="tel:05100000000"
```

شماره خود را جایگزین کنید.

## انتشار در Netlify

1. وارد Netlify شوید.
2. گزینه Add new site را انتخاب کنید.
3. پوشه پروژه را Drag & Drop کنید.
4. سایت منتشر می‌شود.

---

## انتشار در GitHub Pages

1. پروژه را در GitHub آپلود کنید.
2. Settings → Pages
3. Deploy from Branch
4. Branch = main
5. Save

---

## انتشار در Cloudflare Pages

1. پروژه را Import کنید.
2. Framework = None
3. Build Command = خالی
4. Output = /
